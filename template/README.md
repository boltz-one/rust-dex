# {{project-name}}

A GPUI desktop app generated from the boltz template.

## Create a new app

    cargo generate --git https://github.com/boltz-one/rust-dex --subfolder template --name my-app
    cd my-app
    cargo run

Requires the `boltz-*` crates to be published on crates.io. For deep customization or to contribute to boltz itself, clone the full monorepo instead — see its `docs/`.
